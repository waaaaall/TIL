## Japanese Language Pack for Visual Studio Code

## Trailling Spaces

gitの拡張でコミット毎の変更を表示してくてれる。

XDebugが使えるやつ。いちいちvar_dumpとか挟まなくてよくなる。ステップ実行ができる，つよい！  
## PHP DocBlocker
複数行のコメント入力がしやすくなるやつ。
